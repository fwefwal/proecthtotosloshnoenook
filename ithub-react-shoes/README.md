# Практическая работа №4

## Установка и запуск

Клонируйте репозиторий

`git clone https://github.com/daslef/ithub-react-shoes.git`

Перейдите в директорию и установите зависимости

`npm i`

Запустите в соседних вкладках бэкенд...

`npm run server`

...и фронтенд

`npm run dev`

## Задание

Необходимо реализовать страницу добавления продукта, форму добавления продукта и связать ее с бэкендом. Проверить, что добавленный продукт отображается на главной странице и странице категории.

Требования
- страница представляет собой отдельным роут `/create-product`, реализована с использованием `Tanstack Router` (2);
- форма реализована с использованием `React Hook Form`; (3)
- контролы реализованы на основе компонентов `Mantine` (2);
- настроена валидация контролов с использованием встроенных и/или кастомных валидаторов (1);
- отправка данных на бэкенд происходит с использованием хука `useMutation` и функции `create`, описанной в `api/products` (2).
